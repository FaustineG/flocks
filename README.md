# Flocks

This project aims to emulate flocking behaviour in birds.
It is built with vite, vue3, Typescript.

## run project

> npm install
> npm run dev

## run tests

> npm run test:unit

coverage

> npm run test:unit:coverage

mutation

> npm run test:mutation
